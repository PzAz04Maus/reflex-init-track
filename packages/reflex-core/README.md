# Reflex Foundry System Skeleton

Starter scaffold for a custom Foundry VTT system based on Reflex-style initiative and time-track play.

## Included
- Actor types: `character`, `vehicle`, `unit`
- Item types: `weapon`, `gear`, `effect`
- Basic OODA roll helper
- Tick/action cost data model
- Starter sheets and custom document classes
